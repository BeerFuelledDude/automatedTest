1. Install XAMPP
2. Create C:\Behat [for example]
3. Install "Composer" https://getcomposer.org/Composer-Setup.exe
   Point composer to "C:\xampp\php\php.exe" for "Where is PHP.exe located"
4. Put the composer.json file in the Behat folder created
5. Run Composer for CMD, "php composer install"
6. Allow Things to Download
